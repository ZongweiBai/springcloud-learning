### Zipkin

```bash
# download lastest server jar
wget -O zipkin.jar 'https://search.maven.org/remote_content?g=io.zipkin.java&a=zipkin-server&v=LATEST&c=exec'

# start with 'java -jar'
java -jar zipkin.jar
```
